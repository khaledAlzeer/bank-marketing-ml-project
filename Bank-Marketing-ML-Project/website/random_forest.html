📌 What is Random Forest?

Understanding the algorithm

Random Forest is an ensemble learning method that builds multiple decision trees during training and merges their predictions. Each tree is trained on a random subset of the data and features — this randomness reduces overfitting and improves generalization.

The final prediction is determined by majority voting across all trees for classification tasks. The more trees, the more stable and accurate the model becomes.

In this project, we trained a Random Forest with 100 estimators, which gave a significant improvement over the single Decision Tree.

Final Prediction: ŷ = mode { Tree₁(x), Tree₂(x), ..., Treeₙ(x) }

✅ Advantages

  • Reduces overfitting significantly
  • Handles high-dimensional data
  • Robust to outliers & noise
  • Provides feature importance

❌ Limitations

  • Slower than single tree
  • Less interpretable (black box)
  • High memory usage with many trees

💻 Training the Model

How we trained the Random Forest

01

Load Data

# Load preprocessed data
train = pd.read_csv('train.csv')
test = pd.read_csv('test.csv')

print("✅ Data loaded successfully!")
print("Train Shape:", train.shape)
print("Test Shape:", test.shape)
02

Split Data

# Split Features and Target
X_train = train.drop('y', axis=1)
y_train = train['y']

X_test = test.drop('y', axis=1)
y_test = test['y']

print("X_train:", X_train.shape)
print("y_train:", y_train.shape)
print("X_test:", X_test.shape)
print("y_test:", y_test.shape)
03

Train Model

from sklearn.ensemble import RandomForestClassifier

# Train Model
rf_model = RandomForestClassifier(
    n_estimators=100,
    random_state=42
)
rf_model.fit(X_train, y_train)

print("✅ Random Forest trained successfully!")
