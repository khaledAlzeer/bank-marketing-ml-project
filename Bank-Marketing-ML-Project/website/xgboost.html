📌 What is XGBoost?

Understanding the algorithm

XGBoost (Extreme Gradient Boosting) is an optimized distributed gradient boosting library. It builds trees sequentially — each new tree corrects the errors made by the previous one.

Unlike Random Forest which builds trees in parallel, XGBoost builds them one at a time, focusing on the mistakes of the previous iteration. This makes it incredibly powerful for structured/tabular data.

XGBoost dominated ML competitions on Kaggle for years and is widely used in real-world industry applications due to its speed, accuracy, and built-in regularization.

Objective Function: Obj = Σ L(yᵢ, ŷᵢ) + Σ Ω(fₖ)

Where L = loss function  |  Ω = regularization term

✅ Advantages

  • Highest accuracy & AUC
  • Built-in regularization (L1 & L2)
  • Handles missing values natively
  • Fast with parallel computation

❌ Limitations

  • More hyperparameters to tune
  • Less interpretable than LR/DT
  • Slower than simple models

💻 Training the Model

How we trained XGBoost

01

Load Data

# Load preprocessed data
train = pd.read_csv('train.csv')
test = pd.read_csv('test.csv')

print("✅ Data loaded successfully!")
print("Train Shape:", train.shape)
print("Test Shape:", test.shape)
02

Split Data

# Split Features and Target
X_train = train.drop('y', axis=1)
y_train = train['y']

X_test = test.drop('y', axis=1)
y_test = test['y']

print("X_train:", X_train.shape)
print("y_train:", y_train.shape)
print("X_test:", X_test.shape)
print("y_test:", y_test.shape)
03

Train Model

from xgboost import XGBClassifier

# Train Model
xgb_model = XGBClassifier(
    n_estimators=100,
    learning_rate=0.1,
    max_depth=5,
    random_state=42,
    eval_metric='logloss'
)
xgb_model.fit(X_train, y_train)

print("✅ XGBoost trained successfully!")
