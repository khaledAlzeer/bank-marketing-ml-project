📌 Project Overview

What is this project about?

This project applies Machine Learning to a real-world banking problem. Using data from a Portuguese bank's telemarketing campaigns, we built and compared 4 classification models to predict whether a customer will subscribe to a term deposit.

The dataset contains 45,211 records and 17 features including customer demographics, financial info, and campaign history — sourced from the UCI Machine Learning Repository .

Classification Supervised Learning Imbalanced Data SMOTE Binary Prediction
45,211 Records
17 Features
4 Models
88.2% Best Accuracy
